CSS Divoom Ditoo

De opdracht

Je kon uit een aantal verschillende opdrachten kiezen waaronder het maken van een modulair bedieningspaneel, waarvoor ik heb gekozen. Toen dacht ik opeens aan een soort mini computertje die ik op Amazon had zien staan en dacht ik dat is leuk om na te proberen te bouwen. Wat ik zo goed mogelijk heb geprobeerd te doen.

Nieuwe CSS gebruiken

Het was de bedoeling om zoveel mogelijk voor mij nieuwe CSS te gebruiken. Er is namelijk veel CSS die je niet snel of vaak gebruikt. Daarom was het de bedoeling om veel dingen die voor jou nieuw waren uit te proberen. Zo had ik nog niet vaak met gradients gewerkt en wist ik ook nog niet dat je meerdere gradient in een kan plaatsen. Verder had ik ook nog niet met motion paths gewerkt en met box-shadow pixel art gemaakt. Dit heb ik dus allemaal in dit project verwerkt.

Als laatste heb ik ook met de nieuwe :has() selector gewerkt, waardoor dit project nog niet in alle browsers werkt. Op mijn GitHub kun je lezen in welke browsers ik mijn project heb getest.
