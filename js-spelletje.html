Javascript spelletje

De opdracht

De opdracht was om een soort tamagotchi te maken. Ik heb ervoor gekozen om een persoon te maken die je kunt aankleden, koffie kan geven en daarna naar werk kan sturen. Je kunt haar aankleden door kleding uit de kast naar haar toe te slepen of op opgevouwen kleding te klikken. Het idee dat je had moest je gaan bouwen met HTMl, CSS en vanilla JavaScript.

Wat heb ik geleerd?

Ik heb geleerd hoe je met javascript elementen kunt manipuleren. Dit heb ik gedaan met variables, verschillende functions en if else statements.
