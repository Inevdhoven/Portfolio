Rijksmuseum Art Experience

De opdracht

De opdracht was om uit een lijst met user stories te kiezen welk jij graag wilde gaan uitwerken. Ik ben voor de user story van het Rijksmuseum gegaan. Die was als volgt “Als kunstliefhebber, wil ik thuis kunst uit het Rijksmuseum kunnen zoeken en bekijken, zodat ik ten alle tijden van kunst kan genieten.” Met deze user story heb ik een Single Page App gemaakt waar gebruikers allemaal verschillende soorten kunst uit de API van het Rijksmuseum kunnen bekijken.

Hoe ben ik te werk gegaan?

Eerst ben ik gaan kijken naar wat ik allemaal uit de API van het Rijksmuseum kon halen, daarna ben ik gaan schetsen, hoe ik de app eruit wilde laten gaan zien. Daarna heb ik de schetsen uitgewerkt tot een design in Figma. Na het ontwerpen ben ik begonnen met het opzetten van de app in HTML, CSS en JavaScript. Omdat het een Single Page App moest worden moest ik ervoor zorgen om met JavaScript steeds de pagina te veranderen. Dit heb ik gedaan met routie.js, deze library maakt het makkelijk om een router aan je website toe te voegen. Verder heb ik met dit project in ES modules gewerkt en ook met UI-Stack gewerkt door het toevoegen van een loading, empty en error state.
